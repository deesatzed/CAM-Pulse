🔍 Discover

Finds new GitHub and HuggingFace repos worth learning from, then tracks when they change.

🧠 Learn

3-pass mining pipeline extracts reusable patterns, stores them with lifecycle tracking, retrieves by relevance.

  • Classify (free) → overlap assessment → focused LLM extraction
  • 6-factor confidence scoring beyond cosine similarity
  • Co-retrieval links strengthen patterns that work together

cam learn search "middleware chain"

🔨 Build

Injects learned patterns into agent prompts, generates code, and self-corrects on failure.

  • 4 backends: Claude, GPT, Gemini, Grok (or Ollama/MLX-LM locally)
  • Inner correction loop: byte-level restore + re-prompt with violations (up to 3x)
  • "90% coverage" auto-extracted to a hard verification gate

cam create --execute --check "pytest -q"

✔ Verify

Every build checked against real diffs, real tests, real metrics. Failures reported honestly.

  • If no files changed, the run fails — no hallucinated success
  • TruffleHog secret scanning blocks leaked credentials before mining
  • cam forge-benchmark reports "0% lift" when that's the truth

Built to Fail Honestly

These are the things CAM has been explicitly hardened against — because they are easy places for agent systems to become unreliable.

01

Rejects hallucinated success

Other agents say "I updated the files." CAM checks the actual diff. If no files changed, the run is explicitly failed. This is a core architectural decision, not a wrapper.

02

Namespace-safe execution

In fixed-mode, CAM rejects agent output that introduces new top-level source namespaces. --namespace-safe-retry auto-retries with hardened constraints.

03

Preflight contract system

Before attempting risky execution, CAM asks the high-value questions, produces a reusable task contract, and blocks unsafe work. Answers persist across reruns.

04

Honest benchmark reporting

The benchmark harness reports "0% lift" when the fixture corpus doesn't beat baseline. CAM does not overstate results.

05

Documented limits

CAM openly states what does not work yet. --execute is not yet a guaranteed autonomous builder. Local mode is functional but less battle-tested than cloud mode.

06

Full provenance tracking

Every mined methodology records its source repo URL, discovery date, and license type. All repos referenced on this page are open-source projects.
